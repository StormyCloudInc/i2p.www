मुख्य I2P क्लाइंट इम्प्लीमेंटेशन **Java** का उपयोग करता है। यदि आप किसी विशेष सिस्टम पर Java का उपयोग नहीं कर सकते या नहीं करना चाहते हैं, तो समुदाय के सदस्यों द्वारा विकसित और रखरखाव किए गए वैकल्पिक I2P क्लाइंट इम्प्लीमेंटेशन उपलब्ध हैं। ये प्रोग्राम विभिन्न प्रोग्रामिंग भाषाओं या दृष्टिकोणों का उपयोग करते हुए समान मुख्य कार्यक्षमता प्रदान करते हैं।


IMPORTANT: केवल अनुवाद प्रदान करें। प्रश्न न पूछें, स्पष्टीकरण न दें, या कोई टिप्पणी न जोड़ें। भले ही पाठ केवल एक शीर्षक हो या अधूरा लगे, इसे जैसा है वैसा ही अनुवाद करें।

## i2pd (C++)

**वेबसाइट:** [https://i2pd.website](https://i2pd.website)

**विवरण:** i2pd (*I2P Daemon*) C++ में कार्यान्वित एक पूर्ण-सुविधा युक्त I2P client है। यह कई वर्षों से (लगभग 2016 से) उत्पादन उपयोग के लिए स्थिर रहा है और समुदाय द्वारा सक्रिय रूप से बनाए रखा जाता है। i2pd पूरी तरह से I2P network protocols और APIs को लागू करता है, जो इसे Java I2P network के साथ पूर्णतः संगत बनाता है। यह C++ router अक्सर उन प्रणालियों पर हल्के विकल्प के रूप में उपयोग किया जाता है जहां Java runtime अनुपलब्ध या अवांछित है। i2pd में कॉन्फ़िगरेशन और निगरानी के लिए एक अंतर्निर्मित web-based console शामिल है। यह cross-platform है और कई packaging formats में उपलब्ध है — यहां तक कि i2pd का एक Android संस्करण भी उपलब्ध है (उदाहरण के लिए, F-Droid के माध्यम से)।

मैं अनुवाद करने के लिए तैयार हूं, लेकिन मुझे अनुवाद करने के लिए कोई पाठ प्राप्त नहीं हुआ है। कृपया वह पाठ प्रदान करें जिसका आप अनुवाद चाहते हैं।

## Go-I2P (Go)

**रिपॉजिटरी:** [https://github.com/go-i2p/go-i2p](https://github.com/go-i2p/go-i2p)

**विवरण:** Go-I2P एक I2P क्लाइंट है जो Go प्रोग्रामिंग भाषा में लिखा गया है। यह I2P router का एक स्वतंत्र कार्यान्वयन है, जिसका उद्देश्य Go की दक्षता और सुवाह्यता का लाभ उठाना है। यह परियोजना सक्रिय विकास के अधीन है, लेकिन यह अभी भी प्रारंभिक चरण में है और अभी तक पूर्ण-सुविधा युक्त नहीं है। 2025 तक, Go-I2P को प्रायोगिक माना जाता है — इस पर समुदाय के डेवलपर्स द्वारा सक्रिय रूप से काम किया जा रहा है, लेकिन इसके और अधिक परिपक्व होने तक इसे उत्पादन उपयोग के लिए अनुशंसित नहीं किया जाता है। Go-I2P का लक्ष्य एक आधुनिक, हल्का I2P router प्रदान करना है जो विकास पूर्ण होने पर I2P नेटवर्क के साथ पूर्ण संगतता रखता हो।

## I2P+ (Java fork)

**वेबसाइट:** [https://i2pplus.github.io](https://i2pplus.github.io)

**विवरण:** I2P+ मानक Java I2P client का एक community-maintained fork है। यह किसी नई भाषा में reimplementation नहीं है, बल्कि Java router का एक enhanced version है जिसमें अतिरिक्त features और optimizations हैं। I2P+ बेहतर user experience और बेहतर performance प्रदान करने पर केंद्रित है जबकि आधिकारिक I2P network के साथ पूर्णतः compatible रहता है। यह एक refreshed web console interface, अधिक user-friendly configuration विकल्प, और विभिन्न optimizations प्रस्तुत करता है (उदाहरण के लिए, बेहतर torrent performance और network peers की बेहतर handling, विशेष रूप से firewalls के पीछे स्थित routers के लिए)। I2P+ को आधिकारिक I2P software की तरह ही Java environment की आवश्यकता होती है, इसलिए यह non-Java environments के लिए समाधान नहीं है। हालाँकि, उन users के लिए जिनके पास Java है और जो अतिरिक्त क्षमताओं वाला एक वैकल्पिक build चाहते हैं, I2P+ एक आकर्षक विकल्प प्रदान करता है। यह fork upstream I2P releases के साथ up-to-date रखा जाता है (इसकी version numbering में "+" जोड़ा जाता है) और project की website से प्राप्त किया जा सकता है।
