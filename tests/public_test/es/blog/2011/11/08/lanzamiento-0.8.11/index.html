Lanzamiento 0.8.11

By zzz

Como todos habrán notado, el crecimiento de la red sin precedentes a partir del 5 de octubre ha incrementado drásticamente la congestión de la red, especialmente por las tardes (UTC) y los fines de semana. Las dos últimas versiones incluían algunos cambios con los que esperábamos aliviar la presión, pero, por desgracia, estas medidas solo han tenido un éxito modesto. El problema principal es limitar el número de conexiones directas de router a router en la red. Este no es un problema nuevo; hemos estado trabajando en ello durante varios años, con buenos resultados. Sin embargo, el crecimiento reciente volvió a llevarnos al límite.

La versión 0.8.11 incluye varios cambios adicionales para reducir el número de conexiones de router a router y aumentar la capacidad de conexión y de construcción de tunnel. El objetivo, por supuesto, es mejorar las tasas de éxito de la construcción de tunnel y la fiabilidad general. Como siempre, hay algunas correcciones de errores y actualizaciones de traducción.

Damos la bienvenida a todos nuestros nuevos usuarios. Por favor, tengan paciencia mientras trabajamos para mejorar el rendimiento de la red. Depurar problemas de congestión en una red anónima distribuida es un desafío continuo. Por favor, ayuden a mejorar la red reiniciando su router una vez que se haya descargado la actualización.

Los archivos están disponibles en la página de descargas .

DETALLES DE LA VERSIÓN

Cambios importantes

  • Improve peer profile capacity metric to include factors for connection, reachability, country, and router hash, to reduce connection churn
  • Adjust connection limits and timeouts to increase capacity
  • Adjust router info and peer profile expirations to reduce memory usage
  • Rebuild existing tunnels some of the time to reduce connection churn and improve build success
  • Build Executor / Handler thread separation to increase build handling capacity for high-speed routers
  • Bloom Filter optimizations to reduce lock contention for high-speed routers
  • SSU introducer changes

Correcciones de errores

  • Fix expiration of peer profiles

Otros

  • Ukrainian translation updates
